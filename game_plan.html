Game Plan

Game High-Level description

1. Game Title

  • "Catching me !"

2. Description

The player controls a basket at the bottom of the screen. Balls fall from the top, and the goal is to catch as many balls as possible within a set time. Allow mutipule players take turns to catch falling balls,they compete against each other, and the player with the highest score at the end wins.

High-Level Design Statement

1. Theme

  • Colorful cartoon style.

2. Layout

  • Fixed layout.
  • Score and timer display at the top.
  • Player displayed in the top left corner.

3. Color of the balls

  • Use vibrant colors for regular balls.

Player Experience

1. Player Setup

  • Each Player enters their name
  • Players choose the game level: Easy, Medium, or Hard

2. Game Start

  • Player 1 starts the game, controlling the basket using arrow keys
  • Balls begin falling from the top of the screen at random positions.
  • The player moves the basket left and right to catch the balls.

3. Game Play

  • The game runs for a set time(eg. 30 seconds)
  • Player's score increases 1 point by catching a ball.
  • When the timer ends, it's the next player's turn.
  • After Player 1 finishes, the screen displays Player 2's name and score.
  • This process continues for all players.

4. Game End

  • After all players have played, the game compares the scores and highlight the winner.

5. Levels

  • Game has 3 levels: Easy, Medium or Hard
  • Level depends on the speed of falling balls.

Game Elements(Game page)

1. Game Area

  • A container where balls fall and the basket moves.

2. Basket

  • The element players control to catch falling balls.

3. Falling Balls

  • Multiple ball elements that fall from the top of the screen.

4. Player Info

  • An area to display player names and scores during the game.

5. Timer

  • A visible countdown for each player's turn.

6. Control Buttons

  • Buttons allows players to: Pause, Exist and Restart game during the playing.

Game Elements(Welcome page)

1. Game Title

  • Game name
  • Greeting worlds for players

2. Game Description

  • Game Description

3. User Input

  • Input player name
  • Adding player button
  • Level choose button
  • Start game button

4. Display players

  • Display the added players on the screen.

System Actions (functions)

1. Start game

  • Switch from "Welcome page" to the "Game page".
  • Initializes the game for the first player based on the selected difficulty.

2. Drop balls

  • A function that randomly generates balls at the top of the screen and makes them fall at increasing speed based on the difficulty level.

3. Move basket

  • Tracks player input to move the basket left or right.

4. Check for ball catch

  • Counts when a ball has been caught by the basket and increases the player's score.

5. Switch turns

  • Switches the game control to the next player once the current player's turn is over.

6. Update timer

  • Displays and counts down the time remaining for each player's turn.

7. Display scores

  • Shows the current scores of all players at the end of the game.

8. Announce winner

  • Compares scores at the end of the game and announces the player with the highest score.

9. Reset game

  • Resets all values to allow players to play again.
